# 마카샤 시뮬레이터 (Makasha Simulator)

트릭컬 리바이브의 캐릭터 '마카샤'를 소재로 한 웹 기반 클리커입니다

## 플레이 방법
아래 링크를 통해 브라우저에서 바로 실행할 수 있습니다
- 플레이 하기: https://shukketsz9-lab.github.io/makasha-sim/

## 제작자 정보
콘텐츠 제작 및 개발 관련 소식은 아래 채널에서 확인하실 수 잇스빈다
- 제작자 유튜브: https://www.youtube.com/@ShukketszKorea

## 저작권 안내
본 프로젝트는 '트릭컬 리바이브'의 팬 메이드 콘텐츠입니다
사용된 리소스의 저작권은 에피드게임즈(EPID Games)에 있습니다
